# Find-Missing-Number
<h1>Problem statement</h1>
<p>
  Given an array of size N-1 such that it only contains distinct integers in the range of 1 to N. Find the missing element.

# Example 1:

Input:
N = 5<br>
A[] = {1,2,3,5}<br>
Output: 4<br>

# Example 2:

Input:<br>
N = 10<br>
A[] = {6,1,2,8,3,4,7,10,5}<br>
Output: 9<br>
